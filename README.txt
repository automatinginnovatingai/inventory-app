# Inventory App – User Guide  
Version: 2.0.0

The Automating Innovating AI **Inventory App** is a standalone, offline‑capable Windows application designed for precise material, warehouse, and stock‑level management. It includes secure admin onboarding, multi‑warehouse tracking, purchase orders, reporting, and full audit‑ready transaction history.

---

## Key Features

### 🔐 Secure Login & Admin Workflow
- Encrypted password protection using industry‑standard hashing.
- Admin‑only registration (no employee self‑registration).
- Clean onboarding flow: register → login → dashboard.
- Automatic role enforcement and UI adaptation based on admin permissions.

### 📦 Material Management
- Add, edit, and organize materials.
- Track SKUs, categories, units, and cost.
- Manage supplier information and preferred vendors.
- Set reorder points and default units.
- Centralized Material_Info schema shared across all modules.

### 🏷️ Inventory Control
- Manual stock adjustments (add or deduct).
- Scan‑to‑deduct support for barcode/QR workflows.
- Automatic transaction logging with reason codes.
- Real‑time stock level updates across warehouses.
- Multi‑warehouse quantity tracking.

### 🧾 Purchase Orders
- Create and manage purchase orders.
- Track PO status: **Pending → Ordered → Received → Closed**.
- Receive materials directly into inventory.
- Automatic stock updates upon receiving.

### 📚 Inventory Transactions
- Full history of additions, deductions, and adjustments.
- Includes reason, reference number, supplier info, and timestamps.
- Exportable logs for auditing and reporting.
- Designed for transparency and client‑facing accountability.

### 📊 Reporting
- Clean Excel and PDF inventory reports.
- Customizable filenames.
- Print‑ready summaries for office and warehouse use.
- Monthly Excel files generated automatically (optional).

### 🔄 Auto‑Updating
- Automatically checks for new app versions.
- Ensures clients always run the latest stable release.

### 🖼️ High‑Quality App Icon
- Multi‑resolution `.ico` (256 → 16 px) for crisp display on all Windows views.
- Ensures consistent visual weight next to other business applications.

### 🖥️ Friendly Interface
- Modern dashboard with grouped submenus.
- No technical knowledge required to operate.

---

## System Requirements
- **Windows 10 or later**.
- **Microsoft Excel** required for viewing exported reports.
- **~900 MB** of available disk space recommended:
  - SQL Server Express local database storage.
  - Monthly Excel report generation.

---

## License Activation
This app requires a valid **Gumroad license key**.

- You will be prompted to enter your license key on first launch.
- The app verifies your key securely via Gumroad’s API.
- Invalid or revoked licenses restrict access.

> **Note:** Internet access is required for initial license validation only.  
> After activation, the app operates fully offline.

---

## Getting Started
1. Launch the app using the Inventory App icon.
2. Click **Admin Registration** to create your admin account.
3. After registering, click **Admin Login** to sign in.
4. Navigate to the **Inventory Dashboard**.
5. Begin adding materials, warehouses, and managing stock levels.

---

## Navigation & Exiting
- Use the dashboard to access all modules.
- Each module includes a clear **Back** or **Exit** button.
- Closing any screen safely returns you to the dashboard or exits the app.

---

## Tips for Best Use
- Export reports regularly for backups.
- Use strong admin credentials.
- Keep Excel updated for full compatibility.
- Review reorder points monthly to maintain accurate stock planning.
- Use scan‑to‑deduct for faster field workflows.

---

## Troubleshooting

**App won’t launch**  
- Ensure Windows 10+ is installed and Excel is available.

**License key rejected**  
- Verify your Gumroad purchase email and key.

**Export failed**  
- Confirm Excel is installed and the file is not already open.

**Auto‑update not working**  
- Check your internet connection and firewall settings.
