Inventory App - User Guide
====================================
Version: 1.0.0

Welcome to the Automating Innovating AI Inventory App — a streamlined, secure, and intuitive tool 
for managing materials, stock levels, suppliers, purchase orders, and inventory transactions.

Key Features
------------
• Secure Login:
  - Encrypted password protection with industry-standard hashing.

• Material Management:
  - Add, edit, and organize materials.
  - Track SKUs, categories, units, and cost.
  - Manage supplier information and preferred vendors.
  - Set reorder points and default units.

• Inventory Control:
  - Manual stock adjustments (add or deduct).
  - Scan-to-deduct support for barcode/QR workflows.
  - Automatic transaction logging with reason codes.
  - Real-time stock level updates.

• Purchase Orders:
  - Create and manage purchase orders.
  - Track PO status (Pending, Ordered, Received, Closed).
  - Receive materials directly into inventory.
  - Auto-update stock levels upon receiving.

• Inventory Transactions:
  - Full history of additions, deductions, and adjustments.
  - Includes reason, reference number, supplier info, and timestamps.
  - Exportable logs for auditing and reporting.

• Reporting:
  - Clean Excel and PDF inventory reports.
  - Customizable filenames.
  - Print-ready summaries for office and warehouse use.

• Auto-Updating:
  - Automatically checks for new app versions.

• Friendly Interface:
  - Designed for ease of use—no technical knowledge required!

System Requirements
-------------------
• Operating System: Windows 10 or later is required.
• Microsoft Excel must be installed to view exported reports.
• At least 900 MB of available local disk space is recommended:
    - Data is stored using the client's SQL Server Express Database.
    - Monthly Excel files may be generated depending on reporting usage.

License Activation
------------------
This app requires a valid Gumroad license key to unlock full functionality.
• You will be prompted to enter your license key on first launch.
• The app verifies your key securely via Gumroad’s API.
• If the license is invalid or revoked, access will be restricted.
Note: Internet access is required for initial license validation.

Getting Started
---------------
1. Launch the app via the Inventory App icon.
2. Click 'Admin Registration' to create an account.
3. After registering, click 'Admin Login' to sign in.
4. Navigate to the Inventory Dashboard.
5. Begin adding materials and managing stock levels.

Navigation & Exiting
---------------------
• Use the dashboard to access all modules.
• Click the 'Exit' button on any screen to close the app.

Tips
----
• Export reports regularly to back up your records.
• Use strong admin credentials.
• Keep Excel updated for full report compatibility.
• Review reorder points monthly to ensure accurate stock planning.

Troubleshooting
---------------
• App won’t launch: Ensure Windows 10+ is installed and Excel is available.
• License key rejected: Double-check your Gumroad purchase email.
• Export failed: Confirm Excel is installed and the file is not already open.
• Auto-update not working: Check your internet connection and firewall settings.
